# Security Policy

## Reporting a Vulnerability

If you find a security issue — unsafe handling of data, a dependency vulnerability, or anything else — please open an [issue](https://github.com/Sanexxxx777/spread-monitor/issues), or for sensitive reports contact **Aleksandr Shulgin** directly via [@Aleksandr_NFA](https://t.me/Aleksandr_NFA) rather than filing publicly.

## Scope

Spread Monitor is a read-only market-data viewer: it calls public, unauthenticated REST/WS endpoints on exchanges and Dexscreener to display prices, spreads, and funding rates. It does not place orders, does not hold or request API keys/secrets for trading, and does not store user credentials.

## Disclosure

This is a personal open-source project with no fixed SLA, but reports are reviewed as soon as possible.
